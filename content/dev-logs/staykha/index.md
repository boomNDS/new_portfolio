# Staykha Dev Log

## Project Overview

Staykha is a comprehensive dormitory and rental property management system built with:
- **Frontend:** React + Vite + Tailwind CSS
- **Backend:** Elysia (Bun runtime)
- **Database:** PostgreSQL + Supabase
- **Deployment:** Fly.io + Vercel
