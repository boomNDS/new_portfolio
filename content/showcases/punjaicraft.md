# Punjaicraft - Case Study

## Overview

Punjaicraft is a small handmade keychain business that creates unique designs using a 3D printer. I helped build a simple demo website that lets customers preview color collections, choose parts, and understand the ordering steps more easily.

## The Problem

The business owner faced challenges with:
- Customers not being able to visualize final products before ordering
- Difficulty communicating available color options and customization choices
- No clear way to showcase the product creation process

## The Solution

An interactive demo website featuring:
- 🎨 Color collection explorer
- 🔧 Part customization and selection
- 👀 Real-time design preview
- 📋 Clear ordering step guide

## Tech Stack

```
Frontend:     React 18 + Vite
Styling:      Modern CSS
Deployment:   Vercel
```

## Key Features

### 1. Interactive Design Preview
Customers can see how their keychain will look with different color combinations before placing an order.

### 2. Part Selection
Easy-to-use interface for choosing different parts and components of the keychain.

### 3. Ordering Guide
Clear step-by-step explanation of how the ordering process works.

## Result

A simple, effective website that bridges the gap between a small 3D printing business and their customers — making it easier to understand the product and place orders with confidence.
